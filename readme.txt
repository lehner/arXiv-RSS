
arxivrss - Plugin
---------------------------------------

Browse new articles on the preprint server arXiv and import them to
your bibliography.

Author
  * Christoph Lehner (clehner // users.sourceforge.net)

Build instructions:
  * Run build.xml with target jars (default)
  
Changelog:

  * 2009-04-01 - V0.2 - Added search function.

  * 2009-04-05 - V0.3 - Implemented cache, minor bugfixes.

  * 2009-04-25 - V0.4 - Implemented preview of article PDFs.

  * 2009-09-21 - V0.5 - Highlight article if previous version in
    	       	      	database.
